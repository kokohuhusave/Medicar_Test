# medicarTest

### 개발 범위

---

- 시각화: 어플리케이션
- AI 모델: YOLOv8 모델 기반 모델 설계 및 구축

### 프로젝트 내용

---

- 사용자가 핸드폰으로 차량을 촬영 시 손상 부위를 인식하여 분류하는 컴퓨터 비전 프로그램

### 주요 기능

---

- **손상 부위 탐지 -** 차량을 촬영시 손상 부위를 찾아 화면에 표시
- **손상 유형 분류 -** 화면에 표시된 손상 부위의 유형을 분류하여 표시

### 기술 스택

---

**Application**

- AndroidStudio
- Kotlin
- XML

**AI**

- PyTorch
- YOLOv8
